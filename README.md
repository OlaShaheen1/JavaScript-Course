# JavaScript-Course
This repository contains the culmination of my journey through core web technologies, moving from foundational structure to dynamic, interactive applications.

## 🚀 Projects Overview

*   **Portfolio Website:** A responsive showcase of my skills and aesthetic, built with semantic HTML and custom CSS.
*   **Tic-Tac-Toe Game:** A logic-based application demonstrating state management and DOM manipulation via JavaScript.
*   **Calculator:** A functional tool focusing on mathematical logic and event listeners.
*   **Pizza Menu:** A project centered on layout techniques and interactive content display.

## 🛠️ Technical Skillset

*   **HTML5:** Structuring content with accessibility and SEO in mind.
*   **CSS3:** Styling with a focus on responsive design and modern layouts.
*   **JavaScript:** Implementing core programming concepts, API interactions, and user interactivity.

## 💡 Key Takeaways

The primary focus of this collection is the **practical application** of code. By building functional tools from scratch, I have developed a robust workflow for debugging, problem-solving, and full-stack thinking that serves as a springboard for advanced frameworks like React or Vue.js.
